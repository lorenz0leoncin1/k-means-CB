## Overview

This project studies the impact of **initialization** on k-means clustering performance by comparing:
- **k-means** — random initialization
- **k-means++** — D²-proportional seeding

Both algorithms are implemented **from scratch using NumPy only**.

## Experiments

Each experiment runs both algorithms for 50 trials and produces two outputs:
- **Histogram** — distribution of inertia outcomes
- **Boxplot** — stability comparison (spread and variance)

| Experiment | Dataset | km mean | kpp mean | Finding |
|---|---|---|---|---|
| 1 — Synthetic | make_blobs (k=5, std=1.0) | 1466.1 | **1055.5** | k-means++ wins: 28% lower mean, 12.5× lower variance |
| 2 — MNIST [Extension] | MNIST subset (k=10, 784 dims) | 194047.8 | 194101.1 | No advantage — curse of dimensionality |
| 3 — Overlapping [Extension] | make_blobs (k=5, std=3.0) | 6603.5 | 6615.9 | No advantage — ambiguous geometry |
